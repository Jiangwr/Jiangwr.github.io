IDEA编译项目失败 缺少javax包 报错java.lang.ClassNotFoundException:javax.el.ELResolver

May 3, 2018 | 开发经验 | 阅读

因为IntelliJ IDEA 没有导入 servlet-api.jar 这个jar包，需要手动导入支持。

解决方式

  1. 选中项目（在IntelliJ中称为Module）
  2. 点击右键，选择open modual settings
  3. 在弹出的窗口左端选择Libraries
  4. 点击顶端的一个类似加号“+”的图标
  5. 在右端选择第一项（Attach Classes…）
  6. 在弹出的窗口中选择tomcat所在的目录，进入里面的lib目录，寻找servlet-api.jar这个jar包
  7. 选中上述jar包，依次点击OK。
【Maven】 (请使用 -source 8 或更高版本以启用 lambda 表达式)
