May 15, 2018

看书是比较好的学习方式之一。

之前看了在微信工作的大牛写的文章，概括了程序员最重要的三要素:基础 + 数据结构与算法 + 编码素养。

Java 开发涉及到的基础 Java 语言基础、Java 多线程、JVM、Java Web 开发、MySQL、Spring、计算机网络、操作系统等。因此，以此展开分类。

JVM

Java Web

  1. 《深入分析 Java Web 技术内幕》
