Nov 8, 2019

String 类实现的接口

String 类本身被 final 关键字修饰，并且实现了 java.io.Serializable, Comparable, CharSequence 这三个接口。

java.io.Serializable

Serializable 接口没有任何方法和变量。这个接口仅仅用来标示类是否可以序列化。

Comparable

Comparable 接口只有 compareTo(T o) 一个方法。

CharSequence

CharSequence 接口是四个抽象方法和两个 default 方法(JDK1.8 以后接口可以有自己的方法，default 方法、 static 方法)构成。
